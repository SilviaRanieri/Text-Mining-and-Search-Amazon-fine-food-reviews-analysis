	AMAZON FINE FOOD REVIEWS
- Project for text mining and search course -


README
	The project focuses on developing NLP techniques on an amazon review dataset.
	The source code in the Confalonieri_Ranieri_SourceCode.ipynb file is executable using the Colab platform, 
	instead if you want to run the file locally you can install the libraries listed in the requirements.txt file.
	This file is autogenerated by pip freeze.
	
DATA:
	The dataset used for this project can be either download from kaggle (https://www.kaggle.com/snap/amazon-fine-food-reviews) 
	or you can just use the 'Reviews.csv' file present in the 'Data' folder. 
	In that folder you will find also 'Reviews_prep.csv' that contains the same data with the preprocessing phase applied yet.
	
MODEL:
	In 'Keras_Model' folder you can find the trained neural networks models.
